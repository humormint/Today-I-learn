## reset css

- 초기 세팅 미리 해둠

preset.css → 사람에 따라 다를 수 있는 클래스 명

빠짐 없이 import 해주는게 중요!

파일 앞에 언더바(\_) → 어디에 import 되어야 하는 파일

./ → 같은 레벨 → 같은 위치에서 코드 선택

variable 위에 쓰고 reset preset 쓰는게 좋음! → 순서 꼬이면 에러 뜸

→ 새로 파일 생성하면 반드시 추가해 줄 것!
